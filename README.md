## StochasticATM *(In progress)*

Collaborative hackathon project on the application of Neural Differential Equations for the Air Traffic Management (ATM) sector.

Copyright Lucas Berredo, Juan Boudón, Edgar Fernández, 2026, licenced under the EUPL.

Tech specs:
- Three cross-compatible libraries for Dataset generation, Prediction models, and Fuel optimization.
- While there was not enough compute to train large models, the code is easily scalable.
- Overall, the goal was to set the groundwork to apply Neural SDEs to the ATM industry, not to reach production models (though they were surprisingly close). 

Roughly (everyone cross-polinated their ideas into the code), this is the code distribution:

- Dataset generation: Edgar F.
- Prediction model: Lucas B.
- Fuel optimization: Juan B.
